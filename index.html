使用 NetworkManager 工具来管理网络
发表于2023-04-17|更新于2026-03-10|Linux|Linux
使用 NetworkManager 工具来管理网络 其在命令行下对应的命令是 nmcli，要连接WiFi，相关的命令如下： 1）查看网络设备列表 sudo nmcli dev 注意，如果列出的设备状态全部是 unmanaged 的，说明这些网络设备不受NetworkManager管理，你需要清空 /etc/network/interfaces下的网络设置,然后重启. 2）开启WiFi命令 sudo nmcli r wifi on 3）扫描附近的 WiFi 热点 sudo nmcli dev wifi 4）连接到指定的 WiFi 热点 sudo nmcli dev wifi connect “SSID” password “PASSWORD” ifname wlan0 请将 SSID和 PASSWORD 替换成实际的 WiFi名称和密码。
