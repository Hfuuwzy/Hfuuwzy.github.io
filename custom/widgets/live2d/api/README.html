  • ж”ҜжҢҒ жЁЎеһӢгҖҒзҡ®иӮӨ зҡ„ йЎәеәҸеҲҮжҚў е’Ң йҡҸжңәеҲҮжҚў

  • ж”ҜжҢҒ еҗҢеҲҶз»„ еӨҡдёӘжЁЎеһӢ жҲ– еӨҡдёӘи·Ҝеҫ„ зҡ„ еҠ иҪҪеҲҮжҚў

ж·»еҠ жЁЎеһӢ

  • еҚ•жЁЎеһӢ еӨҡз»„зҡ®иӮӨ йҖ’еҪ’з©·дёҫ
    • еӨҡз»„зҡ®иӮӨ з»„еҗҲжЁЎеһӢгҖҒз©·дёҫз»„еҗҲ
    • зҡ®иӮӨж–Үд»¶еӨ№жҢү texture_XX е‘ҪеҗҚ
